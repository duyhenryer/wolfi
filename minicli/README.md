# Minicli Package

Minimalist CLI framework for PHP.

## Package Information

- **Package Name**: `minicli`
- **Version**: See [minicli.yaml](./minicli.yaml) for current version
- **License**: MIT

## Installation

```bash
apk add minicli minicli-bin
```

## Usage

After installation, you can use the `minicli` command:

```bash
minicli --help
```

## Package Components

- `minicli` - PHP library files
- `minicli-bin` - CLI executable command

## Requirements

- PHP 8.2 or higher
